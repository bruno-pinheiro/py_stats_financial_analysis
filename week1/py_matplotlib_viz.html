Visualização de dados com matplotlib¶

Aqui neste notebook vou registrar pores de cdigo para construir visualizaes dos dados das aes do Facebook e da Microsoft utilizando a biblioteca pandas para manipular os dados e matplotlib para visualizar.

In [6]:

fb = pd.read_csv('../data/facebook.csv', parse_dates = ['Date'], index_col = ['Date'])
ms = pd.read_csv('../data/microsoft.csv', parse_dates = ['Date'], index_col = ['Date'])

Visualizar os dados¶

A visualizao dos dados tambm simples. Os objetos do tipo data frame tm uma funo para plotar os seus dados, sem depender de nenhuma outra biblioteca.

A referncia da funo tem mais detalhes sobre o uso, aqui darei apenas alguns simples exeplos.

Por exemplo, para ver a srie temporal do preo "Close".

In [3]:
fb['Close'].plot();
In [ ]:
fb.loc[:, 'Close'].plot();

Todos os dados esto a. Se eu quiser alguns perodos particulares tambm muito fcil. Por exemplo, se eu quiser destacar a variao do primeiro semestre de 2017:

In [7]:
fb['Close'].plot()
fb.loc['2017-01-01':'2017-06-30', 'Close'].plot();

Digamos que eu queira comparar as variaes do Facebook e da Microsoft:

In [21]:
# Criar o grfico
fb['High'].plot(linewidth = 1)
ms['High'].plot(linewidth = 1);

E se eu quiser plotar separadamente:

In [23]:
# Criar o grfico
plt.figure(figsize = (14,5))
subplot(1,2,1)
fb['Close'].plot(color = "dodgerblue")
subplot(1,2,2)
ms['Close'].plot(color = "orange")

Caso eu queira outros formatos basta adicionar a .plot.TipoDePlot(). Para visualizar a distribuio na forma de histogram, por exemplo:

In [24]:
fb.loc[:, 'Close'].plot.hist();

Ou como um boxplot.

In [26]:
fb.loc[:, 'Close'].plot.box();

Se ao invs de um histogram eu quiser um boxplots, tambm simples. A diferena, abaixo, que criarei um subplot para plot os grfico de trs variveis lado a lado.

Para plotar junto, no mesmo grfico, o procedimento um pouco diferente.

In [70]:
data_plot = [fb['Low'], ms['Low']]

plt.figure(figsize = (14,5))
f, (ax) = plt.subplots()
ax.boxplot(data_plot);

Tambm podemos plotar um scatterplot. Por exemplo, para comparar a correlao do volume de transaes de aes do Facebook e da Microsoft.

In [133]:
x = fb['Volume']
y = ms['Volume']
colors = ['blue', 'red']
plt.scatter(x, y, alpha = 0.5, c = colors, s = 3);

Podemos usar um grfico de barras para verificar, por exemplo, quantos dias o colume de transaes de aes do Facebook for maior que o volume de transaes das aes da Microsoft.
